# Module 02 — Foundations of Neuroscience and World Models

## Overview
This module explored how biological systems build internal representations of the external world and use predictive models to guide behavior.

## Topics Covered

- Internal representations
- Predictive processing
- Cognitive maps
- Environmental modeling
- World model theory

## Key Concepts Learned

I learned how the brain continuously predicts sensory input and updates its internal models based on experience. This concept is fundamental in both neuroscience and machine learning.

This module helped me understand how intelligent systems anticipate outcomes and adapt to changing environments.
