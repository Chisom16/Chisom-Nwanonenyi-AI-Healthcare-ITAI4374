# Chisom Nwanonenyi — Computational Neuroscience & Neuro-Inspired AI Portfolio

## About This Repository

Welcome to my academic portfolio for Computational Neuroscience and Neuro-Inspired Artificial Intelligence.

This repository documents my learning journey throughout this course, exploring how biological intelligence, cognitive science, neuroscience, and artificial intelligence intersect to create intelligent systems.

Throughout this course, I studied how the human brain processes information, learns from experience, makes decisions, builds internal models of the world, and inspires next-generation AI systems.

This portfolio contains my module notes, assignments, reflections, research summaries, and project work completed during the course.

Through this course, I developed knowledge and practical understanding in:

- Foundations of neuroscience

- Neurosymbolic artificial intelligence
